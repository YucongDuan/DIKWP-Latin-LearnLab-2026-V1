# DIKWP Latin LearnLab 2026 V1

A standalone offline prototype for Chinese-background learners studying Latin. It focuses on pronunciation, morphology, syntax, evidence-based translation, reading, culture, AI-use transparency, and teacher review.

## Quick start
Open `index.html` in a modern browser. No server, API key, model endpoint, database, or network is required.

Optional CLI:

```bash
python tools/run_latin_plan.py examples/sample_learner.json --out outputs
```

## Boundary
This system is not an official Latin examination, AP score predictor, automated grading engine, ghostwriting tool, or teacher replacement. Outputs are learning evidence and planning aids only.
